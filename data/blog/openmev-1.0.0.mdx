We're happy to announce that OpenMEV `v1.0.0` has been released. This version is
the stable release and comes with lots of performance improvements and bug
fixes.

```bash
npm install @openmev/sdk@latest
# or
npm install @openmev/ethers-provider@latest
```

## Breaking changes

This release includes the following breaking changes.

- Renamed `YCabal` to `OpenMEV`
